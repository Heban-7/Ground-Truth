# Publication Bundle

This directory contains publication-ready drafts and metadata for final release.

## Files

- `publication_manifest.json` (generated snapshot + TODO URLs)
- `dataset_card.md`
- `judge_model_card.md`
- `blog_post_draft.md`
- `community_post_template.md`

## Generate/update manifest

Run:

`python generation_scripts/prepare_publication_bundle.py`

Then fill TODO URLs in:

- `publication_manifest.json`
- `final_submission_checklist.md`
